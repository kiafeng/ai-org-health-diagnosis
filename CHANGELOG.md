## v1.7 · 2026-07-21 — 文档对齐代码 + 公式修正

- **指标计算口径声明**：明确实现采用"均分百分制"口径（单题均分÷6×100），与需求原始"赞成率"口径（选 5-6 分人数占比）不同；所有题目明细双数值输出（均分制得分 + 赞成率/低分率）
- **双支柱健康指数改名**：「组织健康指数」→「双支柱健康指数」（敬业度×50% + 组织能力×50% 等权合成），三处标注"非 McKinsey/Gallup 标准组织健康定义，仅用于内部纵向追踪"
- **健康等级阈值统一 KNX 75/50**：废弃早期 80/60 阈值（≥75 健康 / 50–74 关注 / <50 风险）
- **VP 经理效能矩阵颗粒度 L2→L3**：矩阵从二级部门改为三级部门（与 MES 排名口径统一，消除 skill 内部矛盾）
- **CEO 执行摘要重构为 Executive Brief 叙述式**：结论先行（总体判断 → 最大风险 → 最大优势 → 需关注 BU → 建议 → 关键数据），数字从"主角"变为"附录"
- **移除所有 KPI 卡片 hover**：计算逻辑统一到底部「计算口径说明」注脚区
- **修正 Rule-014 CEO 矩阵 Y 轴公式**：原误写 `(敬业度率 + 组织能力)/2`，改为 `员工敬业度(Q1-14 均分)×50% + 组织能力(Q15-20 均分)×50%`（敬业度率(Gallup 三分类) ≠ 员工敬业度）
- **metric-model.md 标记 DEPRECATED**：5 分制 / 五维模型 / 80-60 阈值文档已废弃，仅作历史参考
