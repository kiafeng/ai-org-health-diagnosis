# 行业基准数据库

## 1. 文件定位

本文件存储 AI 组织健康度诊断 Skill 所需的行业基准数据。

**核心作用**：当 Skill 生成报告时，直接从本文件读取基准数据进行对标，无需再次联网检索。

**使用规则**：
- 所有基准数据均基于 Gallup State of the Global Workplace: 2026 Report 官方数据
- 敬业度分类采用 Gallup 三分类法（敬业 / 不敬业 / 积极不敬业），基于受访者问卷得分直接计算
- 本文件数据截止 2025 年调研周期（2026年4月发布）
- **完整方法论说明**：参见同目录下的 `industry-benchmarks-methodology.md`

## 3. 敬业度分类方法

### 3.1 分类标准

本公司使用 6 分制问卷（1-6 分），严格按 Gallup Q12 对应的 12 道题计算每位受访者均分：

**Gallup Q12 对应题目**（排除自定义题 Q07、Q12）：

| Gallup 原题 | 本问卷题号 | 题目简称 | 所属层级 |
|:-----------:|:---------:|---------|---------|
| GQ1 | Q01 | 工作要求 | L1 基本需求 |
| GQ2 | Q02 | 资源工具 | L1 基本需求 |
| GQ3 | Q03 | 发挥优势 | L2 管理支持 |
| GQ4 | Q04 | 认可表扬 | L2 管理支持 |
| GQ5 | Q05 | 主管关心 | L2 管理支持 |
| GQ6 | Q06 | 鼓励发展 | L2 管理支持 |
| GQ7 | Q08 | 意见重视 | L3 团队归属 |
| GQ8 | Q09 | 使命认同 | L3 团队归属 |
| GQ9 | Q10 | 同事质量 | L3 团队归属 |
| GQ10 | Q11 | 归属支持 | L3 团队归属 |
| GQ11 | Q13 | 进步反馈 | L4 成长发展 |
| GQ12 | Q14 | 学习成长 | L4 成长发展 |

**自定义补充题**（不纳入 Gallup 对标，独立分析）：

| 题号 | 简称 | 所属维度 | 说明 |
|:---:|------|---------|------|
| Q07 | 目标明确 | 管理支持 | 主管说明「什么是好的结果」 |
| Q12 | 坦诚沟通 | 团队归属 | 团队分歧时坦诚沟通、快速对齐 |

### 3.2 三分类标准

| 分类 | 6分制均分 | 百分制 | 对应状态 |
|------|----------|--------|---------|
| 敬业 (Engaged) | ≥ 5.0 | ≥ 83.3% | 强烈同意，工作充满热情 |
| 不敬业 (Not Engaged) | 3.5 - 5.0 | 58.3%-83.3% | 基本同意，但缺乏投入感 |
| 积极不敬业 (Actively Disengaged) | < 3.5 | < 58.3% | 不满，可能影响他人 |

> **计算方式**：每位受访者的 Gallup Q12 题目均分 → 按上表分类 → 统计各类别人数占比 → 与 Gallup 官方公布的区域敬业度率对比。

### 3.3 Gallup Q12 四层金字塔

| 层级 | 名称 | 计算公式 | Gallup 原题 |
|:---:|------|---------|------------|
| L1 | 基本需求 | (Q01 + Q02) / 2 | GQ1 + GQ2 |
| L2 | 管理支持 | (Q03 + Q04 + Q05 + Q06) / 4 | GQ3 + GQ4 + GQ5 + GQ6 |
| L3 | 团队归属 | (Q08 + Q09 + Q10 + Q11) / 4 | GQ7 + GQ8 + GQ9 + GQ10 |
| L4 | 成长发展 | (Q13 + Q14) / 2 | GQ11 + GQ12 |

> 层间差 ≥ 10 分标记为瓶颈层（需求传导断裂）。自定义题 Q07（目标明确）和 Q12（坦诚沟通）不纳入金字塔计算。

## 4. KNX 健康基准（红绿灯判定标准）

> **来源**：组织健康度体检问卷附录 B「健康基准（参考 KNX 常模）」，底层参考 Gallup 全球常模（2023-2024）。

### 4.1 红绿灯阈值表

| 指标 | 🟢 优秀（绿灯） | 🟡 正常（黄灯） | 🔴 预警（红灯） |
|------|:---:|:---:|:---:|
| 单题赞成百分比 | ≥ 75% | 50% - 74% | < 50% |
| 维度得分 | ≥ 75% | 50% - 74% | < 50% |
| 敬业度指数 | ≥ 75% | 50% - 74% | < 50% |
| 组织能力指数 | ≥ 75% | 50% - 74% | < 50% |
| 组织健康度总得分 | ≥ 75% | 50% - 74% | < 50% |
| 低分率 | ≤ 10% | 10% - 15% | > 15% |
| 问卷回收率 | ≥ 80% | 70% - 79% | < 70% |

> **红绿灯说明**：
> - 🟢 **优秀（绿灯）**：团队状态良好，保持现有做法，可作为标杆经验分享。
> - 🟡 **正常（黄灯）**：团队无明显问题，但存在改进空间，建议关注得分最低的 1-2 个维度。
> - 🔴 **预警（红灯）**：团队存在明显短板，需立即启动改进行动，经理人介入辅导。

### 4.2 Gallup 全球常模（2023-2024）

> 此为问卷附录 B 引用的历史 Gallup 数据，与当前 Gallup 2026 Report 的数据有年份差异：
> - 全球敬业度均值：约 23%（严格定义：情感投入 + 主动贡献）
> - 中国敬业度均值：约 15%，低于全球平均水平
> - 科技/互联网行业：20% - 30%，高于全国均值
>
> ⚠️ **注意**：本 Skill 的核心敬业度对标使用 Gallup 2026 Report 最新数据（全球 20% / 中国 20%），上述 2023-2024 数据仅作为历史参考保留。

### 4.3 腾讯 4G 调研参考

基于腾讯年度全员调研（覆盖 2.5-4 万人）的内部维度经验值：

| 维度 | 题号范围 | 通常得分区间 | 说明 |
|------|:---:|:---:|------|
| 基础保障 | Q1-Q2 | 4.0 - 4.2 | 通常得分最高 |
| 管理支持 | Q3-Q6 | 3.6 - 3.8 | 通常为短板 |
| 成长发展 | Q11-Q12 | 3.5 - 3.7 | 通常得分最低 |

> 以上为 4G 内部参考分（6 分制均值），作为各维度得分的经验锚点。

### 4.4 KNX 双视角 5G 常模

| 维度 | 健康标准 | 说明 |
|------|:---:|------|
| 好方向（战略认同） | ≥ 75% | 员工理解并认同组织战略方向 |
| 好工作（流程效率） | ≥ 70% | 工作流程顺畅、工具资源到位 |
| 好领导（管理者有效性） | ≥ 65% | 直接管理者具备有效领导力 |

### 4.5 特别说明

1. **首次调研作为基线**：2026 年首次调研数据作为「基线数据」，重点看「相对位置」而非绝对分数。
2. **团队样本要求**：问卷要求 5 人及以上才出团队报告，5 人以下合并至上级团队以保护匿名性。
3. **跨期对比**：下次体检（2027 年）时，重点对比「同一维度、同一团队」的得分变化，验证改进效果。

## 5. 报告中的对标展示规则

### 5.1 CEO / 管理委员会报告

1. **KPI 仪表盘**：展示敬业度率（本公司 vs 全球 20% / 中国 20%），标注差距
2. **Gallup 敬业度对标章节**：包含 5 项对比
   - ① 敬业员工占比 vs 全球/中国/东亚
   - ② 不敬业员工占比 vs 中国 65%
   - ③ 积极不敬业员工占比 vs 中国 15%
   - ④ vs Gallup 标杆组织 70%
   - ⑤ 分类方法说明
3. **Gallup Q12 金字塔**：四层需求得分 + 瓶颈层标记
4. **自定义题独立分析**：Q07（目标明确）和 Q12（坦诚沟通）单独列出，参考互联网通用管理实践分析

### 5.2 VP / 一级事业部负责人报告

- 事业部敬业度率 vs 全球/中国平均
- 事业部 Gallup Q12 金字塔（含 vs 公司对比）

### 5.3 经理人报告

- 团队敬业度率（如样本量足够）
- 团队 Gallup Q12 各题得分 + 自定义题 Q07/Q12 独立辅导建议
